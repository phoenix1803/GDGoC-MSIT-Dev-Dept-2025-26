# Task 2 - Quiz App

A simple Quiz App for Task 2 that contains basic web development questions. Users can answer multiple-choice questions, and the app calculates the score and shows the results.

## Features
- Multiple-choice questions with **correct/incorrect feedback**  
- **Score calculation** and display of results at the end  
- **Circular progress bar** showing completion  
- On every quiz page: number of questions answered and number of correct answers displayed  
- **Back to Home** button available at the result time  
- **Home page** with a short description of the quiz and a **Start** button  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Run
1. **Download or clone** this repository to your computer.  
2. Navigate to the `code` folder inside `Task-2`.  
3. Open the `index.html` file in any **web browser** (Chrome, Safari, Firefox, etc.).  
4. The quiz will load, and you can start answering questions.
